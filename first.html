Hyper Text Markup Language:

HTML helps you to create webpages and put the content in it. content types can be like text, img, audio, video, tables, forms etc.

It is a markup language. Its not a programming language or a query language . another example of markup language is XML. Markup language basically deals with content creation and holding that content. HTML is like skeleton of the web page and provide basic structure to it.

Language that is written in tags and holding the content can be considered as markup lang. Its not having any logic, it simply have some arrangement. If you want a file to be treated as webpage then you need to make extension as .html.
